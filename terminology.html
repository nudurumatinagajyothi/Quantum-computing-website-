Quantum Terminology

Learn important terms and definitions in the world of quantum computing.

  • Quantum: How much? Refers to the smallest measurable unit in quantum systems.
  • Relay – BP (Belief Propagation): A type of algorithm used to decode quantum errors.
    • Belief Propagation: Each part of a system shares its “belief” about what’s correct, working together to fix mistakes.
    • Relay: Passes information in a layered, smart way, like a relay race where each runner hands off the baton with extra insight.
  • LDPC (Low-Density Parity Check): A method to write information using mostly zeros and few ones — like a secret code that's easy to check for errors.
    • Low-Density: The code doesn’t use too many connections, making it fast and efficient.
    • Parity Check: Checking if things add up correctly, like making sure LEGO pieces fit without gaps.
  • OpenQASM: An imperative programming language designed for near-term quantum computing algorithms and applications.
    • Quantum programs are described using the measurement-based quantum circuit model.
    • Supports classical feed-forward flow control based on measurement outcomes.
